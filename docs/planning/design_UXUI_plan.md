# 나의 제자 (My Jeja) 디자인 설계서 (UI/UX View Architecture) - 최종안

본 설계서는 기획서(`proposal.md`)의 핵심 아이디어인 "학생을 육성하는 역발상 교육 경험"을 시각화합니다.
유저 피드백을 반영하여, **'동물의 숲' 특유의 따뜻하고 아기자기한 캐릭터와 감성은 유지하되, 무대를 숲속 풀밭이 아닌 '아늑하고 퀄리티 높은 교실/학교'로 설정**했습니다.
대학생과 성인층도 '웰메이드 캐주얼 게임'처럼 거부감 없이 즐길 수 있도록 너무 유아틱한 요소는 정돈하고, 게임 UI로서의 세련미를 더했습니다.

## User Review Required
> [!IMPORTANT]
> 무미건조함을 벗고 다시 캐릭터의 생동감과 동물의 숲 감성을 살려냈습니다. 특히 교육 앱이라는 정체성에 맞게 **"햇살이 드는 아늑한 숲속 분교"** 톤앤매너로 배경과 UI를 다듬었습니다. 이 최종안이 구상하시는 "아기자기하되 유치하지 않은, 학교 배경의 앱" 느낌을 잘 잡았는지 검토해 주세요.

---

## 1. 디자인 시스템 (Design System)

### 테마: "햇살 가득한 숲속 교실 (Sunny Forest Academy)"
- **타겟 감성**: 동물의 숲 캐릭터가 등장하는 타이쿤/육성 게임 느낌. 교실이라는 정돈된 공간 안에서 따뜻한 색감과 귀여운 요소들이 어우러지는 힐링 감성.

### 컬러 팔레트 (Color Palette)
- **Primary (자연, 성장)**: 따뜻한 숲의 그린 (Forest Green & Mint, `#6BAA75`, `#B4D4B1`)
- **Secondary (교실, 책상)**: 아늑한 교실의 우드톤 (Warm Oak & Maple, `#DDA77B`, `#8C5E3C`)
- **Background**: 햇살이 비추는 칠판의 부드러운 초록색, 혹은 나무 바닥과 크림 옐로우 벽지.
- **UI 패널 색상**: 부드러운 노란색 계열의 양피지 혹은 코르크 보드 색상 활용.

### 타이포그래피 (Typography)
- UI 수치(레벨, 점수)는 가독성이 뛰어나고 통통 튀는 숫자 폰트 사용.
- 텍스트/채팅 내용은 깔끔한 고딕 베이스에 모서리가 둥근 `나눔스퀘어 라운드`나 `쿠키런체`를 적용하여 친근함 강조.

### UI 모티프 (UI Motifs)
- **School & Nature Mix**: 칠판, 분필, 나무 책상, 공책, 지우개 등의 친숙한 학교 메타포와 사과, 나뭇잎, 곰발바닥 같은 자연의 메타포를 적절히 혼합.
- **Polished Game Status**: 단순히 나무 조각 느낌에서 벗어나, 윤곽선이 깔끔하고 입체감이 살짝 들어간 고품질 캐주얼 게임 UI (예: 테두리가 부드럽게 감싸진 프로필 사진, 게이지 바).

## 2. 화면별 구조 설계 (View Pages Structure)

### 📌 Screen 1: 온보딩 및 입학식 (Onboarding & Placement Test)
- **배경**: 숲속 분교의 아기자기한 교문 앞, 혹은 아침 햇살이 드는 넓은 칠판 앞.
- **입력 폼**: 신입생 전입 신고서 양식(종이 질감)의 팝업 UI. 제자 이름과 개성을 설정.
- **배치고사 진행**:
  - 화면 중앙에 분필로 쓴 듯한 문제 텍스트.
  - 정답을 맞출 때마다 칠판 하단의 '오크통 화분'에서 새싹이 조금씩 자라나는 연출.
  - 선택지 버튼은 지우개나 부드러운 우드 블록 형태를 띰.

### 📌 Screen 2: 홈 화면 (Main Home - 교실 대시보드)
> **동물의 숲 교실 인테리어와 유사한 홈 라운지**
- **배경**: 캐릭터가 서 있는 아늑한 교실 (뒤편에는 사물함이나 책장, 창문 밖으로는 나무 풍경).
- **캐릭터 아바타**: 사용자가 접속하면 책에서 눈을 떼고 웃으며 손을 흔드는 등의 고퀄리티 아이들(Idle) 모션 적용.
- **상태 카드 (학생 수첩 UI)**
  - 좌측 상단이나 하단에 깔끔하게 배치된 학생 수첩 형태의 패널.
  - `민이 (Level 12)`
  - 망각 곡선 게이지는 체력바처럼 명확하게 표현되나, 끝부분에 나뭇잎 디테일이 달려 있음.
- **메인 액션 버튼 (두꺼운 게임형 UI 버튼)**
  - 하단에 큼직한 두 개의 메인 버튼 배치.
  - 🍎 **가르치기 (수업 시작)**: 칠판이나 분필 아이콘 포인트.
  - 📝 **시험 보기 (평가 시간)**: 참 잘했어요 도장이나 백점 시험지 아이콘 포인트.

### 📌 Screen 3: 가르치기 세션 (Teaching Chat Session - 칠판 앞 질의응답)
- **배경**: 진한 녹색의 칠판 텍스처 베이스이거나, 캐릭터가 책상에 앉아 고민하는 교실 풍경 위로 반투명한 채팅창이 열림.
- **채팅 UI (말풍선)**
  - 선생님(유저): 우측 정렬, 따뜻한 코랄/오렌지 색상의 말풍선.
  - 제자(AI): 좌측 정렬, 크림 옐로우 말풍선. 로비에서 보던 제자의 얼굴이 프로필 아이콘으로 등장.
  - **반응형 리액션**: AI 제자가 핵심을 깨달았을 때 제자 프로필 위로 전구(💡) 아이콘이 깜빡이거나 팝업.

### 📌 Screen 4: 함께 시험 치르기 (Joint Exam)
- **화면 분할 (Split View)**
  - 화면 상단: 내 시험지 (양피지 폴리곤 텍스처).
  - 화면 하단: 제자의 '고민하는 텍스트' 영역. 제자의 캐릭터가 이마에 땀을 삐질 흘리거나 연필을 굴리는 애니메이션 동기화.
- **결과 채점 연출**
  - 두 사람의 점수가 화면 가운데로 날아와 커다란 **"합산 85점! 승급 완료!"** 도장(스탬프)으로 쾅 찍히는 다이내믹한 쾌감 연출.
  - 학년 진급 시 제자가 입고 있는 교복에 뱃지가 추가되거나 모자가 바뀌는 보상 시각화.

### 📌 Screen 5: 약점 노트 (Weakness Note / 오답 사물함)
- **컨셉**: '교실 뒤편의 개인 사물함' 
- 열어보면 그동안 틀렸던 약점 개념들이 수첩(노트) 형태로 꽂혀 있음. 
- 복습이 필요한 개념 수첩은 겉표지가 닳거나 먼지가 쌓인 그래픽 이펙트로 표시하여 직관적인 피드백을 제공함.

## 3. 동물의 숲 감성의 세련된 게이미피케이션 
1. **타격감 있는 마이크로 애니메이션**: 
   UI가 화면에 등장할 때 젤리처럼 살짝 튕기는 물리 효과(Bouncy easing)를 주어 게임 특유의 쫀쫀함을 살립니다.
2. **배경의 생동감**:
   단순한 2D 이미지 배경이 아니라, 교실 창밖의 구름이 흘러가거나 햇살 입자가 반짝이는 등 게임 로비를 연상케 하는 웰메이드 생동감을 줍니다.
3. **이모틱콘(이모지) 사용 지양 및 에셋/아이콘 활용 규칙**: 
   UI의 디자인 통일성과 퀄리티를 유지하기 위해 네이티브 텍스트 이모지(👦, 👧 등) 사용을 전면 배제합니다. 대신 `FontAwesome5`와 같은 Vector Icon 라이브러리나 직접 디자인된 이미지 에셋만을 적극 채용합니다.
