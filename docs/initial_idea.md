## STEP 3 — 실현 가능성 × 시장성 × 창의성 매트릭스 평가

> **평가 기준**
> - **실현 가능성(F)**: 6개월 내 MVP 개발 가능 여부 (FastAPI + React-Native 스택 기준), 1~5점
> - **시장성(M)**: 잠재 사용자 규모·지불 의사·경쟁 강도, 1~5점
> - **창의성(C)**: 기존 솔루션 대비 차별성·신선함, 1~5점
> - **종합 점수**: (F × 0.35) + (M × 0.40) + (C × 0.25)

## STEP 4 — Top 3 아이디어 심층 분석

### 🥇 1위: D — "AI 개인 튜터 (24시간 취약점 진단 + 맞춤 학습 경로)"

**핵심 가치**: 학생이 모르는 게 뭔지 모르는 상태(메타인지 결핍)를 AI가 진단하고, 즉각적이고 지속적인 1:1 튜터링 제공

| 항목 | 내용 |
|------|------|
| 타겟 사용자 | 중·고등학생, 대입 수험생, 자기주도 학습자 |
| 핵심 기능 | ① 오답 누적 분석 → 취약 개념 자동 태깅 ② 학생 수준 맞춤 설명 생성 ③ 마이크로 퀴즈로 망각 곡선 대응 ④ 학부모 주간 리포트 자동 생성 |
| 수익 모델 | B2C 구독 (월 9,900원) + B2B 학원 패키지 |
| 기술 스택 | FastAPI + Claude/GPT API + React-Native + PostgreSQL |
| 경쟁 현황 | 뤼이드, 매스프레소 존재 → 차별점: **메타인지 기반 진단 + 학부모 연결** |

**CEO 시각**: 수험생 시장은 지불 의사가 높고, B2B(학원) 연계로 확장성 우수. 학부모 리포트 기능이 재구독률 핵심 레버.

**CTO 시각**: RAG 기반으로 교과서 PDF 인덱싱 → 커리큘럼 정합성 확보. 오답 패턴은 PostgreSQL + 벡터 DB 하이브리드로 처리. MVP 4개월 가능.

### 🥈 2위: F — "수강생 이탈 예측 & 선제적 개입 자동화"

**핵심 가치**: 학원·온라인 플랫폼 운영자가 수강생 이탈 징후를 미리 포착해 매출 손실을 방지

| 항목 | 내용 |
|------|------|
| 타겟 사용자 | 학원 원장, 온라인 교육 플랫폼 운영자, 기업 교육 담당자 |
| 핵심 기능 | ① 수강 패턴(로그인 빈도, 진도율, 오답률) 분석 ② 이탈 위험도 예측 모델 (30일 내 이탈 확률) ③ 자동 개입 메시지 + 상담 예약 트리거 ④ 운영자 대시보드 |
| 수익 모델 | B2B SaaS (학원 월정액, 수강생 수 기반 과금) |
| 기술 스택 | FastAPI + ML 분류 모델 + React (웹 대시보드) |
| 경쟁 현황 | 국내 전무 수준 → 선점 효과 기대 |

**CEO 시각**: B2B라 LTV가 높고 영업 사이클이 짧음. "이탈 1명 방지 = 월 수십만 원 매출 보전" 메시지로 ROI 직관적.

**CTO 시각**: 초기엔 rule-based 이탈 탐지 → 데이터 누적 후 ML 고도화. 기존 LMS API 연동 커넥터가 개발 난이도 핵심.

| 핵심 기능 | ① 실시간 질문 분류 & 자동 답변 ② 이해 못한 학생 비율 실시간 시각화 ③ 강의 후 AI 요약 노트 자동 생성 ④ 후속 복습 문제 자동 출제 |
| 수익 모델 | B2B 대학/기업 라이선스 |

## STEP 5 — 최종 방향 결정 (CEO × CTO 합의)

### 결론: **아이디어 D + F 통합형 솔루션**

> "AI 개인 튜터"와 "수강생 이탈 예측"을 하나의 플랫폼으로 통합한다.
> - **학생 측**: 24시간 AI 튜터가 취약점을 진단하고 맞춤 학습 경로를 제시
> - **운영자 측**: 학습 데이터를 기반으로 이탈 위험 학생을 선제 포착, 개입
> - **학부모 측**: 자녀 학습 현황 리포트 자동 수신

### 솔루션 가칭: **"EduSync AI"**
> *"모든 학생의 학습 흐름을 끊기지 않게 연결한다"*

| 관점 | 핵심 메시지 |
|------|-----------|
| 학생 | 내가 모르는 게 뭔지 AI가 먼저 알려준다 |
| 교사/강사 | 학생 이탈 전에 알 수 있다 |
| 학부모 | 아이의 학습 상태를 매주 리포트로 받는다 |
| 기관 운영자 | 이탈률을 줄여 매출을 지킨다 |
